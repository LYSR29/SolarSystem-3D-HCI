# SolarSystem-3D-HCI
A bespoke 3D Solar System simulation developed in Processing (P3D), featuring a bespoke gesture-based control system via Computer Vision. This project explores real-time orbital mechanics and Human-Computer Interaction :)

🇬🇧 System Engineering Student or anything | HCI & Computer Vision Enthusiast.
🇲🇽/🇨🇴 Aqui se trabaja con logica, el código limpio (o no tan limpio :v)  y no crear mamadas pedorras wey.



### 🇬🇧 The Oxford Perspective
A sophisticated 3D rendering engine built on **Processing (P3D)**. This bespoke simulation integrates **Computer Vision** to bridge the gap between celestial mechanics and intuitive user interfaces. It's not just a map; it's an exploration of how we interact with complex data through natural gestures.

### 🇲🇽/🇨🇴 No soy latina pero trato de entender la vibra weyes
Este proyecto no es cualquier cosa. Aquí le metimos:
* **Visión Artificial:** Usamos la cámara para detectar movimiento. Si mueves la mano, el sistema hace zoom (EN PROCESO DE MEJORA). ¡Una elegancia de la de francia!
* **Matemática Aplicada:** Full lógica con senos, cosenos y álgebra booleana para que los planetas no salgan volando por ahí.
* **Código con Propósito:** Estructurado con POO (Clases) para que sea fácil de entender y no sea un "espagueti" de código.
